# 🪄 Gemini Gem 部署版 · 家庭财务复盘官

把本「家庭财务管理系统」打包成一个可在 **Google Gemini Gems** 上运行的助手。
你每月只需上传两个 Excel（**月度资产负债表** ＋ **开支流水**），即可自动得到完整的
**月度财务复盘报告**、**财务健康度评分（0–100）** 与 **FIRE 进度测算**。

## 📁 文件清单

| 文件 | 用途 |
| --- | --- |
| `01-Gem指令-粘贴到Instructions.md` | 复制其中代码块全文，粘贴到 Gem 的 **Instructions（指令）** 框 |
| `02-知识文件-财务分析方法论.md` | 作为 **Gem 知识（Knowledge）** 上传（公式 / 评分档位 / 参数） |
| `03-知识文件-月度复盘报告模板.md` | 作为 **Gem 知识（Knowledge）** 上传（报告输出格式） |
| `04-部署指南.md` | 详细的部署步骤与每月使用方法 |
| `../templates/` | 三个输入模板（资产负债表 / 开支流水 / 收入），可直接套用或参考 |

## ⏱️ 30 秒上手

1. 打开 **gemini.google.com → 左侧 Gems → 新建 Gem**。
2. **指令**：粘贴 `01-Gem指令-粘贴到Instructions.md` 代码块内的全部文字。
3. **知识**：上传 `02-知识文件-财务分析方法论.md` 和 `03-知识文件-月度复盘报告模板.md`。
4. 保存后，**每月**新建对话上传你的两个 Excel，发送一句：
   > 分析 2026-05 的家庭财务，生成完整月度复盘报告。

> ⚠️ 你的每月数据请在**对话中上传**（不要放进 Gem 知识库）；Gem 知识库只放上面两个静态文件。

更多细节见 [`04-部署指南.md`](./04-部署指南.md)。
