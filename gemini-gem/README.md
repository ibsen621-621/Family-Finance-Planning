# 🏛️ Gemini Gem 部署版 · 家族办公室首席风控官（FO-CFO）

把本系统升级为一个在 **Google Gemini Gems** 上运行的「家族办公室首席风控官」。
它以**投行 IBD 风控视角**服务高净值实控人：每月上传资产负债表与开支流水（及补充输入），
即可得到 **IBD 研报格式的月度风控研报**、**高净值五维风控评分（0–100）**，以及
**LDI 安全底盘覆盖率 / SRR 退休红区 / DIME 跨代际防火墙 / 大额节点资本截流率** 测算与**极限压力测试**。

> 本版本刻意废弃“月度储蓄率 / 50-30-20 / 6 个月应急金 / 静态 4% 法则”等中产早期模板，
> 改用 Morningstar 3.9% 安全红线、负债驱动（LDI）与序贯回报风险（SRR）框架。

## 📁 文件清单

| 文件 | 用途 |
| --- | --- |

| `../templates/` | 输入模板（资产负债表 / 开支流水 / 收入 / 家办补充输入） |

## ⏱️ 30 秒上手

1. 打开 **gemini.google.com → 左侧 Gems → 新建 Gem**。
2. **指令**：粘贴 `01-Gem指令-粘贴到Instructions.md` 代码块内的全部文字。
3. **知识**：上传 `02-知识文件-财务分析方法论.md` 和 `03-知识文件-月度复盘报告模板.md`。
4. 保存后，**每月**新建对话上传数据（资产负债表、开支流水、`家办风控-补充输入` 等），发送一句：
   > 以家办风控官视角分析 2026-05 的家族财务，先做数据脱水，再出 IBD 研报。

> ⚠️ 你的每月数据请在**对话中上传**（不要放进 Gem 知识库）；Gem 知识库只放上面两个静态文件。

更多细节见 [`04-部署指南.md`](./04-部署指南.md)。
